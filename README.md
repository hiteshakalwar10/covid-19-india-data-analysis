# COVID-19 India Data Analysis using Python

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the COVID-19 India dataset using Python. It analyzes case trends, recovery rates, active cases, deaths, and state-wise statistics through data visualization.

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset

COVID-19 India Dataset

## Analysis Performed

- Data Loading
- Data Cleaning
- Top 10 States by Confirmed Cases
- COVID Cases Over Time
- Recovery Rate Over Time
- Active vs Recovered Cases
- Deaths Over Time
- Correlation Heatmap
- Outlier Detection
- Top 10 States by Deaths

## Skills Demonstrated

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Time Series Analysis
- Pandas
- Matplotlib
- Seaborn
